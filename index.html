SplitNow · Shared expenses with friends

Angular · TypeScript · PWA

Progressive web app to split expenses among friends. It automatically calculates who owes what to whom and generates a ready-to-share message for WhatsApp.

  • Add participants.
  • Register each expense and who was involved.
  • Automatic balance summary (who pays whom).
  • Shareable closing message.
  • Hosted on Vercel with a custom domain.

Role: full development with Angular (frontend, calculation logic, UX).

DollarTracker · Real-time dollar rates

React · Vite · PWA

I built a PWA to display real-time USD exchange rates in Argentina. It embeds trusted external iframes to keep data always up-to-date and is optimized for fast loading with a native-like experience.

  • Real-time updates via trusted iframes.
  • Installable with manifest and service worker.
  • Optimized for mobile and desktop use.
  • Hosted on Vercel with a custom domain.

Role: full build (React, Vite, PWA setup, and Vercel deploy).
