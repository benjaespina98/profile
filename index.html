Proyectos destacados

SplitNow · División de gastos entre amigos

Angular · TypeScript · PWA

Aplicación web progresiva para dividir gastos en una juntad. Calcula automáticamente quién debe cuánto, a quién, y genera un mensaje listo para copiar y pegar en WhatsApp.

  • Alta de participantes.
  • Registro de cada gasto y quién participó.
  • Balance final automático (quién le paga a quién).
  • Resumen de cierre en texto compartible.

Rol: desarrollo completo (frontend, lógica de cálculo, UX).
